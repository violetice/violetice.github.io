一只爱好广泛的程序猿

你好，我是Violet_ice紫冰。欢迎来到我的小作坊。

是一名生于 98 年初的小青年。当年学习不好，没本事上大学，上了个 2+3，专业是会计电算化。考了三年的会计从业资格证，没考出来。随后就取消了这个证[1]。

后来表哥去学了 php，大概半年后我也报了培训班。如今一直在从事全栈开发。

名称的来源，其实与 Minecraft 有关啦。早年前碰过 alpha 版本，应该也都知道的，只能放方块和破坏，其他什么都干不了。 时间来到 2014 年，在优酷看到了岷叔，此后的三年就一直在 MC 圈子混，也模仿了这一群人的名称风格。

还因为 MC 而去自学了 java，虽然没学会~。但掌握了一门语言之后，在回来自学 java 就很简单了。

我还是一名自媒体创作者，当然主要还是游戏，还是以跑跑卡丁车为主的。
